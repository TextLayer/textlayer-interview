POST /v1/threads/chat

Send natural language queries about financial data and receive AI-generated responses with SQL queries and results.

Request Body:

{
  "messages": [
    {
      "role": "user",
      "content": "What is the total revenue for all customers in 2018?"
    }
  ]
}

üóÑÔ∏è Database Schema

The financial database contains the following tables:

  • financial_data: Main fact table with financial metrics and amounts
  • account: Chart of accounts (Revenue: 400, 4000, 4010, 4020; Expenses, Assets, etc.)
  • customer: Customer dimension (West, Midwest, Southeast, Southwest regions)
  • product: Product dimension (Product lines, categories)
  • time: Time dimension (2018M01, 2018M02, etc. format)
  • version: Version dimension (ACT=Actual, BUD=Budget, FC1=Forecast)
  • time_perspective: Time perspective (BASE, YTD calculations)
